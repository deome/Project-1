Inline vs Block tags: Inline tags lack size and other attributes used in block tags. Block tags allow for CSS and class behavior in HTML.

Notes from Project 1

div can be used to create boxed structures. Box structures can possess a class in CSS. Boxes can be visible or invisible. Everything is a box. Boxes can be assigned a CSS class, or CSS classes can be assigned to box elements. Boxes can also be controlled by adjusting box parameters in CSS.
