Lesson 1.1

The Web

The Web is made up of linked HTML/etc. documents. Supposedly "invented" in early 90's, though I'm sure the web developed in the 80's.

Components of the Web

The Web consists of individual users/browsers, the internet, and servers. These components communicate through the HTTP protocol.

HTML

HTML consists of text content, markup, references to other documents, and links to other pages. HTML markup is made up of tags.

DOM

The document object model is a wireframe or mold for markup languages, including HTML, that allows them to incorporate hierarchy into a document (via "tags" in HTML)
